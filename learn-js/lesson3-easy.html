Lesson 3: If / Else 🧠

1 The Idea

If / Else is just the computer asking a YES or NO question.

Think of it like this:

Your mom asks: "Did you finish your homework?"

If YES → "You can play games!" 🎮

If NO → "Go finish it first!" 📚

That's it! The computer asks a question. If the answer is YES, it does one thing. If NO, it does something else.
