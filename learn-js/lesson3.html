Lesson 3: If / Else! 🧠

1 Real Life Example

You already use if/else every day! You just don't know it:

If it's raining 🌧 → take an umbrella

Else (not raining) ☀ → wear sunglasses

If you're hungry 🤤 → eat food

Else (not hungry) 😊 → keep playing

In code, it works the exact same way!

2 How to Write It

if (something is true) {
    // do this
} else {
    // do that instead
}

if 💬 "Hey computer, CHECK something"

( ) 💬 "Here's what to check"

{ } 💬 "Do the stuff inside these curly brackets"

else 💬 "If the check FAILED, do this instead"

3 Try It: Age Check

Let's check if someone is old enough to drive:

let age = ???;

if (age >= 18) {
    console.log("You can drive! 🚗");
} else {
    console.log("Too young to drive! 👶");
}

>= means "greater than or equal to"

So age >= 18 means "is age 18 or more?"

4 Try It: Secret Password

Let's make a password checker! The secret password is "pizza".

let guess = ???;

if (guess === "pizza") {
    console.log("Correct! You may enter! 🔓");
} else {
    console.log("Wrong password! 🔒");
}

=== means "is this EXACTLY equal to?"

We use three equals signs === to CHECK.

We use one equals sign = to STORE (like in variables).
