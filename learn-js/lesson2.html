Lesson 2: Variables! 📦

What you'll learn: How to store information using variables — like boxes that hold stuff!

What is a Variable? 🤔

Imagine you have labeled boxes in your room:

📦 A box labeled "myName" — inside it you put "Jiten"

📦 A box labeled "myAge" — inside it you put 10

📦 A box labeled "favoriteColor" — inside it you put "blue"

A variable is just like that — a named box that stores information!

Step 1: Create Your First Variable

We use the word let to create a variable:

let myName = "Jiten";
console.log(myName);

let = "Hey computer, I want to create a box!"

myName = the label on the box

= means "put this inside the box"

"Jiten" = what goes inside the box

Step 2: Different Types of Variables

Variables can hold different types of stuff:

let playerName = "Jiten";    // Text (called a "string")
let age = 10;               // Number
let score = 0;              // Number
let isPlaying = true;       // True or False (called a "boolean")

console.log("Player: " + playerName);
console.log("Age: " + age);
console.log("Score: " + score);
console.log("Playing: " + isPlaying);

"text" = words go in quotes (called a String)

10 = numbers have NO quotes (called a Number)

true / false = yes or no (called a Boolean)

+ joins text together (like glue!)

Step 3: Changing a Variable

You can change what's inside a variable — like swapping what's in your box!

let score = 0;
console.log("Score at start: " + score);

score = 10;
console.log("Score after round 1: " + score);

score = 25;
console.log("Score after round 2: " + score);
Notice: We only use let the FIRST time. After that, just use the name to change it!

Step 4: Math with Variables! 🧮

You can do math with number variables — just like a calculator!

let apples = 5;
let oranges = 3;

let total = apples + oranges;
console.log("Total fruits: " + total);

let difference = apples - oranges;
console.log("Difference: " + difference);

let multiplied = apples * oranges;
console.log("Multiplied: " + multiplied);

+ = Add

- = Subtract

* = Multiply

/ = Divide

What You Learned Today 🎓

let creates a new variable (a named box)

= puts a value inside the variable

Variables can hold text ("hello"), numbers (42), or true/false

+ with text joins them together

+ - * / with numbers does math

You can change a variable after creating it
