Lesson 4: Functions! 🤖

1 The Idea

A function is like a robot helper that does a job for you.

Imagine you have a robot. You teach it ONE job:

🤖 "When I say GREET, say Hello!"

Now every time you say "GREET", the robot says "Hello!"

You taught it once, and it can do it again and again!

That's a function! You teach the computer a job, then you can use it whenever you want.

2 Try It: Greeting Robot

We taught the robot to say hello. Click the button to call the robot!

🤖

Robot's job: Say "Hello, friend!"

In code, it looks like this:

function greet() {
    show "Hello, friend!"
}

greet();   // Call the robot!

function = "I'm teaching the robot a job"

greet = the robot's name

greet() = "Hey greet robot, do your job!"

3 Giving the Robot Information

You can also give the robot some information to use!

🤖 "When I say GREET and give you a name, say Hello to THAT person!"

You say: GREET("Jiten") → Robot says: "Hello, Jiten!"

You say: GREET("Mom") → Robot says: "Hello, Mom!"

Same robot, different name each time!

Click a name to give it to the robot:

🤖

Robot's job: Say hello to whoever you give it!

In code:

function greet(name) {
    show "Hello, " + name + "!"
}

greet("Jiten");   // Hello, Jiten!
greet("Mom");     // Hello, Mom!

The name inside ( ) is like a slot — you put different things in each time!
