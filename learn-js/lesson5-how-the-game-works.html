1 The Variables (What the game remembers)

The game needs to remember things. That's what variables are for!

let lives = 3; // How many hearts left
let streak = 0; // How many correct in a row
let timeLeft = 60; // Seconds remaining
let level = 1; // Your current level
let xp = 0; // Experience points
let currentAnswer = 0; // The correct answer

2 Making a Random Question (Functions + Variables)

The game needs to make a new question every time. Here's how:

// Step 1: Pick two random numbers
let num1 = randomNumber(1, 10);
let num2 = randomNumber(1, 10);

// Step 2: Calculate the correct answer
let currentAnswer = num1 + num2;

// Step 3: Show the question
show( num1 + " + " + num2 + " = ?" );
💡 Connection to Lesson 4: randomNumber() is a function (a robot helper)! You give it a minimum and maximum number, and it picks a random one.

3 Checking the Answer (If/Else!)

When you type an answer, the game uses if/else to check it:

let guess = what you typed;
let currentAnswer = the real answer;

if (guess === currentAnswer) {
    // YES! Correct!
    score = score + 10;
    streak = streak + 1;
    show("Correct!");
} else {
    // NO! Wrong!
    lives = lives - 1;
    streak = 0;
    show("Wrong!");
}
💡 Connection to Lesson 3: This is just a YES/NO question!
If the guess matches the answer → add points
Else → lose a life

4 Streak Bonus (If inside If!)

When you get many correct in a row, the game gives you bonus points:

if (guess === currentAnswer) {

    streak = streak + 1;

    if (streak >= 5) {
        points = 30; // SUPER STREAK!
    } else if (streak >= 3) {
        points = 20; // STREAK!
    } else {
        points = 10; // Normal
    }

    score = score + points;
}
New trick: else if

Sometimes you need MORE than 2 choices. else if adds more options:

if (streak is 5+) → 30 points
else if (streak is 3+) → 20 points
else → 10 points

It's like: "If it's hot, wear shorts. Else if it's warm, wear a t-shirt. Else wear a jacket."

5 The Countdown Timer (New concept!)

The timer counts down every second. Here's how it works:

let timeLeft = 60;

// Every 1 second, do this:
everySecond(function() {
    timeLeft = timeLeft - 1; // Remove 1 second
    show(timeLeft + "s"); // Show on screen

    if (timeLeft <= 0) {
        endGame(); // Time's up!
    }
});
New concept: setInterval

It's like setting an alarm that rings every second.
Each ring: subtract 1 from timeLeft.
When timeLeft hits 0: game over!

6 Level System (Variables + If/Else)

The level system tracks your XP (experience points):

let level = 1;
let xp = 0;
let xpNeeded = 50;

// When you answer correctly:
xp = xp + 10;

// Check if you leveled up:
if (xp >= xpNeeded) {
    level = level + 1; // LEVEL UP!
    xp = 0; // Reset XP
    show("LEVEL UP!");
}
💡 It's just variables + if/else!
Every correct answer adds XP to the xp variable.
If xp reaches the goal → level goes up and xp resets!

7 Sound Effects (Bonus!)

Sounds are made by telling the computer to vibrate the speaker at a certain speed:

// Make a beep sound
playTone(523, 0.1); // frequency 523, for 0.1 seconds

// Correct sound = 3 beeps going UP
playTone(523); // low beep
playTone(659); // middle beep
playTone(784); // high beep 🎶

// Wrong sound = 2 beeps going DOWN
playTone(200); // medium buzz
playTone(150); // low buzz 💔
Higher number = higher pitch.
523 = middle C note on a piano. 784 = a higher note. 150 = a low buzz.

9 Saving Scores (Leaderboard)

When the game ends, we save the score so it's there next time you open the game:

// Save to the browser's memory
localStorage.setItem("leaderboard", scores);

// Load it back later
let scores = localStorage.getItem("leaderboard");
localStorage is like a secret drawer in your browser.
You put something in it, close the browser, come back later, and it's still there!
That's how the leaderboard remembers your scores.
