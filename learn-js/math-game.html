❓ How to Play
🎮The Goal
Answer as many math questions as you can before the timer runs out or you lose all your lives!
⏱Timer
You have 60 seconds to answer as many questions as possible. Be quick!
❤Lives
You start with 3 lives (hearts). Each wrong answer loses 1 life. If you lose all 3, the game ends!
⭐Points & Scoring
✅ Correct answer = +10 points
🔥 3 in a row = +20 points (Streak bonus!)
💥 5 in a row = +30 points (Super streak!)
⏩ Skip = 0 points (but no life lost)
📈Levels
You start at Level 1. Earn XP by answering correctly!
Every time the XP bar fills up, you LEVEL UP!
Higher levels = bigger numbers = more challenge!

XP needed per level:
Level 1 ➔ 2: 50 XP
Level 2 ➔ 3: 80 XP
Level 3 ➔ 4: 120 XP
Level 4 ➔ 5: 170 XP
And it keeps going!
